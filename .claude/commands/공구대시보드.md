## 데이터 소스
친한스토어 어드민 REST API 직접 호출:
- Base URL: `https://api.store.friendly-pharmacist.com`
- 인증: Bearer JWT 토큰 (Header: `Authorization: Bearer <token>`)
- API 1: `GET /api/commerce-be/v1/sellers` — 셀러 목록
- API 2: `GET /api/admin/v1/group-purchases/plans/{id}` — 공구 플랜 상세

## 주요 함수
- `connectWithToken()` — 토큰 입력 → API 테스트 → localStorage 저장
- `apiCall(endpoint)` — Bearer 토큰 포함 fetch wrapper
- `fetchSellers()`, `fetchPlans()`, `fetchPlanDetail(id)` — API 호출
- `normalizePlanDetail(raw)` — API 응답 → 통합 데이터 구조
- `renderStats()` — 통계 카드 4개 (총 공구, 진행중, 셀러, 상품)
- `renderPlanChart()` — 공구별 상품 현황 CSS 바 차트
- `renderCommissionChart()` — 커미션율 비교 차트
- `renderTable()` — 정렬/검색/필터 테이블
- `openDetail(planId)` — 공구 상세 모달 (상품, 옵션별 가격/커미션)
- `exportExcel()`, `exportCSV()` — SheetJS 기반 내보내기
